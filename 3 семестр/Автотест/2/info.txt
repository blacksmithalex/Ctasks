Файл tree.o содержит откомпилированные заготовки для работы с бинарным неупорядоченным деревом, в каждой вершине которого хранятся два целых значения int value, balance. Заготовки содержат функции для создания дерева и распечатки поддеревьев. Файл tmain.срр иллюстрирует работу этих функций. Прочитайте комментарии в файлах tmain.сpр и tree.h. Используя данные заготовки напишите программу решения следующей задачи:

Назовем балансом поддерева число, равное разности между количеством всех вершин поддерева и количеством его концевых вершин (не имеющих обоих потомков). Баланс пустого дерева есть 0. Требуется проставить во всех вершинах балансы соответствующих поддеревьев и распечатать полученное дерево. После этого требуется ввести с клавиатуры число Х и удалить из дерева одну любую вершину со значением value = Х. Если вершина имеет одного потомка, то удаление происходит естественным образом, если два, то вместо данной вершины прицепляется ее поддерево с наибольшим количеством вершин (при равенстве - правое), а другое поддерево уничтожается. После удаления балансы корректировать не надо. Полученное дерево также распечатать.
