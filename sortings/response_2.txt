При решении следующей задачи нужно реализовать отдельную функцию для сортировки массива чисел, функцию для проверки массива на упорядоченность, функцию для чтения массива из файла и функцию для генерирования случайного массива указанной длины. Можешь использовать аналогичные тем, что написал для предыдущей задачи. 

Необходимо реализовать сортировку, которая группирует положительные элементы в начале массива, а отрицательные в конце так, чтобы и положительные, и отрицательные элементы внутри своей группы были упорядочены по возрастанию. Назови сортировку произвольно. 

Нужно составить тест для проверки трудоемкости алгоритма сортировки. Для этого реализуется текущая сортировка и сортировка с использованием библиотечной функции qsort. Для заданного N генерируется неупорядоченный массив из N чисел, который сортируется всеми тремя методами. N задается с консоли, исходный массив генерируется в файл input.txt, итоговый отсортированный массив выводится в файл output.txt. Ограничение на модуль каждого из элементов массива сделай в отдельную константу. 

Замеряется время работы каждой сортировки (например, с помощью функции clock() или любых других подходящих).
